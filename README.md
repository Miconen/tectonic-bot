# OSRS Clan Discord Bot

Install:

```
git clone https://github.com/Miconen/tectonic-bot
cd tectonic-bot
```
Set bot token in .env file
```
BOT_TOKEN=BOT_TOKEN_HERE
```
Install:
```
npm install
```
Development build:
```
npm run start
```
Production build:
```
npm run build
npm run serve
```


### Discordx
Boilerplate and template from https://github.com/oceanroleplay/discordx-templates/
